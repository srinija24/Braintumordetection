# Installation
```
python -m venv myenv

myenv\Scripts\activate

```

# File Creation

Create a file named requirements.txt and place the packages that required to run this project

```text

Flask
Werkzeug
numpy
Pillow
opencv-python
tensorflow

```
# Installing All packages at once

```code
pip install -r requirements.txt
```
